# 项目规范索引（INDEX）

PigeonDeck 的颗粒化规范。每条规则一个文件，本索引每行一条：标题 + 一句话钩子。新增或变更规范时，建/改对应单规则文件并同步更新本索引（见根 `CLAUDE.md` 的自更新规则）。

- [git-workflow](git-workflow.md) —— 什么该/不该提交，提交与分支习惯。
- [ignore-policy](ignore-policy.md) —— 哪些文件/目录被排除以及为什么。
- [file-management](file-management.md) —— 目录归属与文件命名（kebab-case）。
- [interaction-invariants](interaction-invariants.md) —— 页面内 UI 交互不变量（浮层再点即关/拖拽关闭/编辑面 Ctrl+Enter·Esc/选中抑制框/打印隐藏/动画），改 UI 必读。
