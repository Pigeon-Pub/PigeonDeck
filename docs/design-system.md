> **定位**：PigeonDeck 的完整视觉语言参考。设计令牌从 pigeonlib 照搬，控件配方从 `preview/` 组件画廊提取。编码阶段以此文件为 UI 实现的单一真相源。

### 1.4 圆角

| 元素 | 值 | Token |
|------|----|-------|
| 控件（药丸形） | `9px`（输入框/下拉/按钮/分段/数值控件用 `999px` 全圆） | `--r-ctl` |
| 卡片/浮层 | `12px` | `--r-card` |

### 1.5 动效

| 档位 | 时长 | 缓动 | 适用场景 |
|------|------|------|----------|
| 快 | `130ms` | `ease-out` | hover 高亮、tooltip、按钮内反馈 |
| 中 | `190ms` | `ease-out` | 浮层展开/收起、面板弹出、位号卡片展开 |
| 慢 | `260ms` | `ease-soft` | 拖拽松手归位、参考线消失、大面积过渡 |

**全局规则**：
- 动画仅使用 `transform` 和 `opacity`，禁止触发 layout
- 组件内禁止私自定义时长/缓动——统一引用 token
- 不弹跳、不炫技

## 2. 图标

| 项目 | 详情 |
|------|------|
| 来源 | [Lucide Icons](https://lucide.dev)（MIT） |
| 风格 | 轻量线性 SVG，stroke-width 1.5-1.6px |
| 交付 | 内联 SVG 字符串（见 `pigeon-components.js` I 对象） |
| 工具盘 | 20px |
| 面板按钮 | 16px |
| 颜色 | 跟随 `--ctx`，激活态使用 `--c1` |

**原则**：优先使用 Lucide 已有图标，无合适匹配时手绘补充。所有图标保持一致 stroke-width。

---

## 3. 硬性禁止

- ❌ emoji（所有图标用 Lucide SVG 或手绘 SVG）
- ❌ 写实鸽子、花哨插画、复杂纹理
- ❌ 玻璃拟态、重渐变
- ❌ 竖线装饰、卡片套卡片
- ❌ 过重边框、过多说明文字

---

## 5. UI 组件规格

### 5.1 Logo 悬浮球 `.pd-ball`
- 尺寸：42px 直径（正圆形）
- 背景：`--c1`（邮政金）
- 内部：白色鸽子 SVG 线稿（22px）
- 阴影：`0 4px 14px -4px rgba(120,84,20,0.55), 0 1px 3px rgba(0,0,0,0.18)`
- 位置：默认右下角，距边缘 16px
- 交互：点击展开/收起；长按拖拽

### 5.2 单列纵向工具盘 `.pd-toolbar`
- 布局：单列纵向，仅图标，无文字
- 容器：药丸胶囊形（`border-radius: 999px`），内边距 5px，按钮间距 3px
- 表面：`--csf` 底 + `--cbd` 边框 + `--sh-up` 阴影
- 激活态：外描边 `--c1-edge`
- 列宽：由 Logo 金底球 42px 决定

### 5.3 工具盘按钮 `.pd-tbtn`
- 尺寸：38px 圆形（Logo 按钮 42px）
- 默认：透明底 + `--ctx` 图标色
- Hover：`--chv` 底
- 激活：`--c1` 图标色 + `--c1-soft` 底 + `--c1-edge` 边框
- Tooltip：hover 时在按钮旁显示（12px 字号，`--ctx` 底色 + `--csf` 文字）

### 5.4 合并撤销/重做 `.pd-undoredo`
- 形态：横向药丸（42px × 23px），无中缝
- 左半 = 撤销，右半 = 重做
- Hover 半区：`--chv` 底 + `--c1` 图标色
- 图标：stroke-width 1.9（加粗加深）

### 5.5 面板 `.pd-surface`
- 容器：`--csf` 底色 + `--cbd` 边框 + `--sh-up` 阴影
- 圆角：`--r-card` (12px)
- 宽度：280-360px（按内容自适应）
- 标题：`Fraunces + 思源宋体`，600 字重

### 5.6 底栏 `.pfoot`
- 布局：`space-between`，上下 `--cbg` 底色 + 上 `--cbd` 分隔线
- 左侧：位号 · 元素类型 · 位置(px)，11px `--ctx2`
- 右侧：删除按钮（圆形 iconbtn 危险色）+ 保存按钮（药丸 primary）
- 无关闭按钮（面板靠 Esc / 点击外部 / 保存关闭）

### 5.7 文本按钮 `.pd-btn`
- 形态：药丸形（`border-radius: 999px`），最小高度 30px
- 默认：`--csf` 底 + `--cbd` 边框
- Primary：`--c1` 底 + 白字
- Danger：`--cd` 底 + 浅底字
- Ghost：透明底 + 透明边框，hover 显 `--chv`
- 图标：15px，与文字间距 6px

### 5.8 图标按钮 `.pd-iconbtn`
- 尺寸：28px 圆形
- 默认：透明底 + `--ctx2` 图标色
- Hover：`--chv` 底 + `--ctx` 图标色
- Danger hover：`--cd` 图标色 + 红色浅底

### 5.9 输入框 `.pd-input`
- 形态：药丸形（`border-radius: 999px`），padding 6px 13px
- 表面：`--cbg` 底 + `--cbd` 边框 + `--ctx2` 文字色
- Focus：`--c1-edge` 边框 + `--c1-soft` 外光晕 (0 0 0 3px)

### 5.10 多行文本框 `.pd-textarea`
- 形态：圆角矩形（`border-radius: --r-ctl`），非药丸
- 自适应高度：`field-sizing: content`，最小高度 38px
- 文字色：`--ctx`（保持可读）

### 5.11 下拉 `.pd-select`
- 形态：药丸形 + 右侧小圆球箭头（`--chv` 底 + chevron `--ctx2`）
- 去原生箭头（`appearance: none`）
- 选择器内部 padding-right 34px 为箭头留空

### 5.12 数值控件 `.pd-num`
- 形态：大胶囊（`border-radius: 999px`）+ 内部嵌套小圆胶囊 `+/−`
- 结构：`[输入值] [单位px] [小圆胶囊: +−]`
- 单位：小圆胶囊左邻，`--ctx2` 色
- 小圆胶囊：`--chv` 纯色底、无阴影，`+−` 为 Lucide 图标（13px）
- 按钮：宽 20px、高 19px，reset `padding:0`

### 5.13 分段控件 `.pd-seg`
- 形态：药丸容器（内边距 2px），内含等宽选项
- 选中项：`--csf` 底 + `--ctx` 字色 + `--sh` 阴影
- 图标模式 (`.pd-seg.icons`)：padding 5px 11px，图标 15px
- 文字装饰模式 (`.pd-seg.deco`)：B/I/U/S 字形切换

### 5.14 颜色控件 `.pd-color`
- 一行三件：色块圆(30px) + 色值药丸 + 取色器按钮
- 色块：圆形（`border-radius: 50%`），透明色显示棋盘格底
- 棋盘格：`linear-gradient(45deg,#c8c8c8 25%,transparent 25%...)` 10px 格
- 取色器：30px eyerop 图标按钮，hover 变 `--c1`

### 5.15 调色盘（展开）
- 完整取色器 + 局部取色推荐（祖先链取 computed color，去重按频率取前 7）+ RGB 三通道 + 透明度滑杆
- 7 个推荐色块：圆形、铺满卡片宽度、`flex:1` + `aspect-ratio:1`

### 5.16 开关 `.pd-switch`
- 尺寸：38px × 21px，全圆（`border-radius: 999px`）
- 关闭态：`--cbd` 底 + 白圆 thumb
- 开启态：`--c1` 底
- 动画：130ms ease-out

### 5.17 分段显示图标 `.pd-seg-display`
- 4 图标选项：block / flex / grid / inline（各配中文 tooltip）
- 同 `.pd-seg.accent.icons` 配方

### 5.18 分类导航 `.pd-nav`
- 窄竖向（46px 宽），圆形选中样式
- 每项：图标圆(28px) + 下方标签(10px)
- 选中：`--c1` 字色 + `--c1-soft` 圆底 + `--c1-edge` 内描边
- 变更角标：左竖条(2px `--c1`) + 右上计数点(`--c1` 底白字)

### 5.19 设置面板 `.spanel`
- 头部：`Fraunces` 15px 标题 + 关闭

- 行：`.pd-srow` 无逐行横线，靠留白 + 分组标题分隔

### 5.20 变更说明 `.pd-diff`
- 精简格式：`旧值 → 新值`（右值 `--c1` 色加粗）

## 7. 修改栏智能切换

| 元素类型 | 修改栏默认控件 |
|----------|--------------|
| 文本元素 | 文字内容 + 字体/字号/字重/颜色/对齐/行高/字距/margin/padding |
| 图片 | 尺寸(宽/高) + 边框/圆角 + 替换按钮 |
| 视频 | 尺寸(宽/高) + 边框/圆角 + 替换按钮 |
| 按钮/容器 | 背景色 + 圆角 + 边框 + 阴影 + 不透明度 + margin/padding |
| 其他（陌生元素） | 按 computed style 自动列出最相关控件，带「自动」角标 |

## 8. 双入口单源规则

属性控件由统一注册表 `FIELDS` 管理（`preview/pigeon-components.js`）。修改栏和高级样式区引用**同一个 `<pd-prop field="...">` 原件**——改一处两处同步。修改栏 = 按元素类型挑的高频子集，必为高级样式全集真子集。

---

## 9. 亮/暗双主题

实现方式：Shadow DOM 根元素 `data-theme` 属性切换，所有 CSS 变量跟随。

```
:root, [data-theme="light"] { ... }   ← 亮色（默认）
[data-theme="dark"] { ... }           ← 暗色
```

切换入口：设置面板 → 通用 → 主题（日/月图标）。
