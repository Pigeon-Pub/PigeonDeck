# 删除布局策略与设置提示设计

## 背景

当前删除快捷键会直接移除目标元素，并仅在标注记录中保留 `deleted` 标记。删除后的原节点只存在于撤销命令闭包中，导致覆盖层、图片导出和清空操作无法共享删除现场：删除框会消失，图片中缺少明确的删除批注，清空也无法恢复被删除内容。

本次同时在设置面板加入删除布局策略，并为所有设置项补充悬浮解释。

## 目标

- 删除元素后始终保留金色标注框和位号。
- 图片导出始终包含删除框、位号和本地化的“删除 / Delete”卡片。
- 清空时恢复被删除的原节点；撤销清空重新应用删除，重做清空再次恢复。
- 用户可选择删除后保留原布局空间或让页面自动重排。
- 设置面板中的每个设置行都有中英文悬浮解释。

## 非目标

- 不改变 Delete 快捷键的生效范围和输入框聚焦保护。
- 不让设置切换追溯修改已经发生的删除。
- 不新增自制 tooltip 浮层或调整设置面板整体布局。

## 删除布局策略

新增持久设置 `deletionLayout`，取值如下：

- `preserve-space`：默认值。保留原节点的布局位置，但使其不可见、不可交互、不可聚焦。覆盖层仍以原节点的实时矩形绘制标注框。
- `reflow`：将原节点从 DOM 脱离，页面正常重排。覆盖层使用删除瞬间保存的文档坐标绘制标注框，因此可能与重排后的内容重叠；设置提示必须明确说明这一点。

删除动作执行时读取当前设置，并把采用的策略写入该删除记录。之后修改设置只影响新的删除，不改变已有删除。

## 删除现场

删除标注保存删除前的文档坐标和所用布局策略。新增会话内删除现场注册表，以标注 ID 为键保存：

- 原 HTMLElement 对象；
- 原父节点和后继节点；
- 隐藏前需要恢复的内联状态与可访问性状态。

注册表仅在当前 tab 会话内存在，符合项目不跨刷新持久化标注的约定。

`preserve-space` 不创建克隆占位节点。它保留原节点并临时设为透明、禁用指针事件和焦点，避免克隆造成重复 ID、事件丢失或 flex/grid 尺寸偏差。恢复时原样还原节点此前的状态。

`reflow` 从 DOM 脱离原节点。恢复时优先插回原后继节点之前；若后继节点已不在原父节点，则追加到原父节点末尾。

## 数据流

### 删除、撤销与重做

1. 选中元素按 Delete。
2. 记录删除前文档矩形、当前布局策略和运行时现场。
3. 按策略隐藏或脱离节点。
4. 将带删除元数据的标注写入 Store，并推入 History。
5. 撤销恢复原节点和原标注；重做重新应用该条记录保存的策略。

### 覆盖层

- `preserve-space` 优先读取仍在 DOM 中的隐藏节点矩形。
- `reflow` 使用删除记录中的文档矩形减去当前滚动偏移，绘制稳定的视口矩形。
- 两种策略都使用现有金色标注框和位号，不新增视觉样式。

### 图片导出

- 删除记录使用保存的文档矩形参与截图范围和覆盖层绘制。
- 即使删除记录没有说明文字或样式变化，也必须生成卡片。
- 卡片类型按导出语言显示 `Delete` 或 `删除`。

### 清空

清空命令先保存当前 Store 快照及删除现场，然后恢复所有隐藏或脱离的原节点，再清空标注 Store。撤销清空时恢复 Store 并重新应用各条删除策略；重做清空再次恢复节点并清空 Store。

## 设置界面

在“交互”分区新增“删除布局”文字分段控件。为避免中文在紧凑设置行内换行，界面只显示短标签，完整语义保留在悬浮解释中：

- “保留”：默认选中，悬浮解释为删除内容不可见但保留布局空间，标注框不会与重排内容冲突。
- “重排”：悬浮解释为页面会收缩重排，删除框可能覆盖移动后的内容。

所有设置行通过原生 `title` 提供本地化解释。解释挂在整行，用户悬浮标签、空白区域或控件附近均可看到。快捷键动作的解释由快捷键定义中的 `descKey` 提供，避免面板内重复维护。已有图标按钮的具体 `title` 保持不变。

## 兼容性与异常处理

- 旧设置缺少 `deletionLayout` 时由默认设置自动补为 `preserve-space`。
- 删除现场只保证当前会话内恢复；页面刷新后标注本身也会按现有规则清空。
- 原父节点在恢复前已被页面脚本移除时，不猜测新的插入位置；保留记录供导出，但无法恢复该节点到页面。
- 无效 selector 不影响删除记录的导出，删除元数据是导出定位的真相源。

## 测试与验收

- 设置模型：默认值、旧设置合并、两种策略持久化。
- 删除：保留空间模式隐藏并可恢复原节点；重排模式脱离并原位恢复；设置切换不改变已有删除。
- 覆盖层：两种策略均显示标注框，滚动后位置正确。
- 导出：删除记录进入截图范围，绘制框和位号，并生成中英文删除卡片。
- 清空：恢复两种删除；撤销清空重新删除；重做清空再次恢复。
- 设置面板：分段控件显示“删除布局 / 保留 / 重排”，全部设置行和快捷键行具有本地化解释。
- 发布版本：所有产品版本来源一致为 `1.0.1`，changelog 正确归档本次更新。
- 门禁：目标单测、`npm run typecheck`、`npm test`、`npm run i18n:check` 和受影响的图片导出/清空 E2E。
