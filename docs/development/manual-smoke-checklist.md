# PigeonDeck 手动冒烟测试清单（V1）

面向真实网站的人工冒烟清单，覆盖 Playwright 自动化无法可靠验证的部分（尤其
`chrome.tabs.captureVisibleTab` 截图、`file://` 页面、Popup 站点禁用等需要真实
浏览器上下文的能力）。每完成一次发布候选（RC）走查一遍。

## 测试环境

- 浏览器：Chrome 最新稳定版 / Edge 最新稳定版（各走查一遍）。
- 加载扩展：
  1. `npm run build` 生成 `dist/`。
  2. 打开 `chrome://extensions`（或 `edge://extensions`）。
  3. 打开右上角「开发者模式」。
  4. 点「加载已解压的扩展程序」→ 选择项目 `dist/` 目录。
- 测试页面：至少覆盖一个内容型站点（含文本/图片/表单）、一个 SPA、一个
  `file://` 本地 HTML、一个 PDF 链接。
- 亮/暗主题各走查关键项一遍。

---

## 1. 安装与首次运行
- [ ] 加载 `dist/` 后扩展出现在扩展列表，无报错（点「错误」按钮应无内容）。
- [ ] 首次安装自动打开「安装说明 / onboarding」页面。
- [ ] 任意普通网页右下角出现悬浮球。

## 2. 悬浮球与工具盘
- [ ] 悬浮球显示正常（Logo 白描边鸽），hover 有反馈。
- [ ] 点击展开工具盘：7 按钮（Logo / 移动 / 复制文本 / 复制图片 / 撤销重做 / 清空 / 设置），顺序与图标正确。
- [ ] 各按钮 hover 出 tooltip。
- [ ] 长按悬浮球拖拽可移动位置。
- [ ] 刷新页面后悬浮球位置恢复到上次拖拽处。
- [ ] 展开即进入批注模式（无独立批注按钮、无高亮）。

## 3. 批注（元素）
- [ ] 批注模式 hover 元素出高亮框 + 元素标签。
- [ ] 单击元素出批注面板，填写保存后出现位号圆 + 标注框。
- [ ] 点位号展开卡片，再点收起。
- [ ] 通过修改栏/高级样式改字号、颜色、背景等，页面元素实时变化，保存后卡片出现「调整项」。
- [ ] 右键位号 → 菜单 → 删除；UI 消失。
- [ ] 删除后再建标注，编号不复用（编号只增不重排）。

## 4. 直接编辑与内联富文本
- [ ] 双击文本元素进入 contentEditable，可直接改文字。
- [ ] 出现 Word 式富文本浮条：加粗/斜体/下划线/字号等生效，选区保持正确。
- [ ] 图片/视频元素触发替换弹层，可替换 URL；本地大图（>1MB dataURL）只活内存不落库。

## 5. 区域框选
- [ ] 批注模式空白处长按（约 300ms）出金色实时框，拖拽成区域。
- [ ] 松手弹区域批注面板，填说明保存后出现持久区域框 + 位号。

## 6. 移动模式
- [ ] 切移动模式，hover 未选中元素出圆角高亮框。
- [ ] 单击选中元素出选中框 + 八向缩放句柄，拖句柄可改宽高。
- [ ] 点住元素本体拖动出 translate 预览。
- [ ] 拖到与其它元素对齐处出参考线（白/黑反色），有吸附。
- [ ] 按住 Alt 拖动为自由移动（无吸附、无参考线，出 free 提示）。
- [ ] 面板 +/- 胶囊调整选中粒度（组件块层级）。
- [ ] 同一元素多次移动，最终只合并为「初始→最终」（见复制文本输出）。

## 7. 撤销 / 重做
- [ ] 工具盘合并药丸：左半撤销、右半重做，按可用性正确禁用/启用。
- [ ] 展开态 Ctrl/Cmd+Z 撤销、Ctrl/Cmd+Shift+Z 重做生效。
- [ ] 收起态快捷键不生效。
- [ ] 批注/样式/区域/移动/清空各操作均可撤销恢复。

## 8. 复制文本（重点）
- [ ] 造多条标注（含元素批注 + 样式修改 + 移动 + 区域）后点「复制文本」。
- [ ] 结果弹窗出现，正文含 `[Page Context]` / `[Global Editing Rules]` / `[Operations]`。
- [ ] 弹窗可滚动预览、快切语言（en / zh_CN / 跟随界面）、下载 `.md`、再复制。
- [ ] **粘贴到编辑器（VS Code / Codex 输入框）验证任务清单结构可被 AI 执行**：
  - [ ] 每条操作有 `#编号` + Type（如 `Annotation + Style Modification + Move`）。
  - [ ] 样式修改以 Changes 表呈现；内容修改分流正确。
  - [ ] 含「不要硬编码 top/left，优先 flex/grid/gap/margin/order」类全局规则。
  - [ ] 移动只呈现初始→最终，不含中间步。
- [ ] 默认导出英文；设置切中文/跟随后再复制语言正确。

## 9. 复制图片（真实 Chrome 关键项 —— 自动化无法验证）
> `captureVisibleTab` 在 Playwright 环境挂起，只能真机冒烟。
- [ ] 造标注（含区域/移动）后点「复制图片」。
- [ ] 长页面滚动拼接成单张长图，无明显撕裂/错位（横向滚动页可能拉伸，已知妥协）。
- [ ] 叠加正确：编号圆、标注框、区域框、移动幽灵框 + 连线，配色与 UI 一致。
- [ ] 弹窗「复制」按钮 → 粘贴到聊天/图片编辑器成功。
- [ ] 弹窗「下载」按钮 → 下载 PNG 成功。
- [ ] 水印设置开/关生效。
- [ ] `fixed` 定位元素在长图中可能重复（已知妥协，确认可接受）。

## 10. 清空
- [ ] 点「清空」贴按钮弹确认层；点外部/取消关闭、不清空。
- [ ] 确认清空 → 所有位号/框消失。
- [ ] 清空后 Ctrl+Z 可整体恢复（清空本身可撤销）。
- [ ] 清空后新建标注编号从 #1 重置。

## 11. 设置面板
- [ ] 点「设置」出面板，含 4 分区（通用 / 交互 / 输出 / 帮助）导航切换。
- [ ] 亮/暗主题切换即时生效（Shadow DOM `data-theme`）。
- [ ] 界面语言：搜索式选择器可搜索/筛选/选择，切换后界面文案更新。
- [ ] 导出语言（英文 / 跟随界面 / 指定）持久。
- [ ] hover 标签开关、长按时长、历史上限步进等各项修改后重开面板仍保留。
- [ ] 关闭按钮 / Esc / 点外部均可关闭面板。

## 12. Popup（工具栏图标弹窗）
- [ ] 点扩展图标出 Popup。
- [ ] 全局禁用开关：关闭后当前页 reload，悬浮球消失；重开恢复。
- [ ] 站点禁用开关：禁用当前站点后 reload 生效，其它站点不受影响。
- [ ] 禁用列表展示已禁用站点，可移除。
- [ ] `file://` / PDF 页面在 Popup 显示对应「不支持 / 需开启文件访问」提示。

## 13. 右键菜单
- [ ] 页面右键出现「用 PigeonDeck 快速标注」菜单项。
- [ ] 点击后进入批注并对右键目标元素开面板。

## 14. 特殊页面
- [ ] `file://` 本地 HTML：需先在扩展详情页开启「允许访问文件网址」，之后悬浮球正常注入、全流程可用。
- [ ] PDF 页面：Popup 显示「不支持 PDF」提示，不注入 UI。

## 15. 状态生命周期
- [ ] 刷新页面：可定位的标注/区域/移动恢复；找不到目标的元素不乱改页面，仅轻提示，任务记录保留。
- [ ] 同一 URL 视为同一会话（页面键 = 完整 URL）。
- [ ] 关闭标签页后重开该 URL：会话已清理（新建为空）。

## 16. 主题
- [ ] 亮色主题下所有 UI（工具盘/面板/卡片/弹窗/参考线）配色正确、可读。
- [ ] 暗色主题下同样正确，无对比度问题。

---

### 已知妥协（非缺陷，走查时确认可接受）
- 横向滚动页复制图片可能拉伸。
- `position:fixed` 元素在长图拼接中可能重复出现。
- 自动写剪贴板在异步截图管线后手势失效，改由弹窗「复制」按钮触发（可靠）。
- `file://` 访问需用户在扩展详情页手动开启（浏览器 per-extension 设置，扩展无法自动开启）。
