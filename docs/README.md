# 文档导航

`docs/` 是项目文档的统一入口。根目录 `CLAUDE.md` 只保留常驻规则，专题信息按任务从这里加载。

- [product/](product/README.md)：当前产品规格与行为定义。
- [design/](design/README.md)：视觉系统和 UI 历史裁决。
- [development/](development/README.md)：维护指南、已完成实施基线和验证清单。
- [conventions/](conventions/README.md)：持续生效的仓库规范。
- [history/](history/README.md)：开发交接与用户反馈原始记录。
- [archive/](archive/README.md)：已被当前规格取代的早期蓝图。

新增文档应进入最贴近其用途的分类目录，不要把专题细节写回 `CLAUDE.md`。
