PigeonDeck

Annotate any web page and hand a ready-to-run task list to your AI.

Look for the pigeon in the bottom-right corner of any page.

Quick start

How it works

1

Open the toolbar

Click the floating ball in the corner to expand the tools. It opens straight into annotation mode.

2

Click an element

Single-click any element to drop a numbered pin and write a note. Long-press and drag to box a region.

3

Edit, style and move

Double-click text to rewrite it, tweak styles in the card, or switch to Move mode to reposition components with snapping.

4

Copy and hand off

Hit Copy as Text for an AI-ready task list, or Copy as Image for an annotated screenshot.

What you can do

Annotate
Numbered pins with notes on any element.
Region select
Long-press and drag to mark an area.
Direct edit
Rewrite text and tune styles inline.
Move & snap
Drag components with edge and center guides.
Copy as Text
Generate an AI-ready task list.
Copy as Image
Export a single annotated long screenshot.

Pin the hero heading and ask for shorter copy, recolor a button, then drag the sign-up card up to align with the nav. One Copy as Text later, you have a structured task list — page context, global rules, and every change — ready to paste into Codex or any coding AI.
